PRATICANDO HTML E CSS

Está é uma página de praticas em HTML e CSS.

1. HTML

HTML é uma linguagem de marcação semântica, ou seja, é em HTML que é feita toda parte de estruturação de determinada página web

2. CSS

CSS é uma linguagem de estilização, é na CSS que é feita toda a parte de forma e estilo de determinada página web
